# Lucide React

**Versão**: `lucide-react@0.454.0`

## Onde é usado
Iconografia consistente em todo o site, com tree-shaking automático via Vite.

## Aplicações no projeto
- **Hero**: `ArrowRight`, `Play`.
- **Problem**: `Brain`, `RefreshCw`, `Network`.
- **Services**: `LayoutDashboard`, `Database`, `GitBranch`, `Zap`, `Shield`, `GraduationCap`, `ArrowUpRight`.
- **Cases**: `ArrowUpRight`.
- **Testimonials**: `Quote`, `Star`, `ChevronLeft`, `ChevronRight`.
- **Pricing**: `Check`, `ArrowRight`.
- **FAQ**: `Plus` (gira 45° ao abrir).
- **CTA**: `ArrowRight`, `CheckCircle2`.
- **Footer**: `Instagram`, `MessageCircle`, `Mail`, `ArrowUp`.
- **Header / WhatsApp**: `Menu`, `X`, `ArrowRight`, `MessageCircle`.

Tamanhos padronizados: `14` (chips/inline), `16` (botões), `18–22` (cards principais).
